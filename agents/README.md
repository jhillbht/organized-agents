## 🚀 Quick Agent Import

### **Import All Claudia Agents**
```bash
# Copy your exported agents to Claudia
cp agents/claudia-exports/* ~/claudia/agents/

# Restart Claudia to load new agents
open -a Claudia
```

### **Set Up Enhanced Agents**
```bash
# Install enhanced agent dependencies
cd agents/enhanced-agents
npm install

# Configure for your environment
npm run configure

# Test all agents
npm run test-agents
```

```javascript

  name: "Beginner Bridge Agent",
  model: "claude-sonnet-4",
  systemPrompt: "You are a friendly guide helping developers learn parallel agents...",
  tools: ["explanation", "progress-tracking", "encouragement"],
  costTier: "standard",
  maxTokens: 4000

```

### **Coordinate Multiple Agents**
```javascript
// Agent coordination script
const coordination = {
  agents: [
    { name: "planning", role: "orchestrator" },
    { name: "research", role: "information-gatherer" },
    { name: "implementation", role: "code-creator" },
    { name: "testing", role: "quality-assurer" }
  ],
  workflow: {
    phase1: "planning coordinates all agents",
    phase2: "research and testing work in parallel",
    phase3: "implementation uses research results",
    phase4: "testing validates implementation"
  }
};
```
