## 🎯 **Quick Start: Your First Agent Success**

*From zero to working parallel agents in 24 hours*

### **Prerequisites**

- [Claudia Desktop App](https://github.com/getAsterisk/claudia) (GUI for Claude Code)
- Node.js 18+ and npm/yarn
- Git and basic command line familiarity
- An adventurous spirit! 🚀

### **Installation**

```bash
# Clone the repository
git clone https://github.com/jhillbht/organized-agents.git
cd organized-agents

# Run the magic setup script
npm run setup

# Verify everything works
npm run verify
```

### **Your First Agent**

```bash
# Start with the Beginner Bridge Agent
cd examples/01-single-agents/beginner-bridge
npm start

# Follow the interactive tutorial
# In 15 minutes, you'll have your first agent running!
```
