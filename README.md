## ğŸ“¦ Installation

### Prerequisites

- **Rust** (latest stable version)
- **Bun** (for package management)
- **Git** 

### Quick Start

```bash
# Clone the repository
git clone https://github.com/Organized-AI/organized-agents.git
cd organized-agents

# Install dependencies
bun install

# Start development server
bun run tauri dev

# Build for production
bun run tauri build
```
